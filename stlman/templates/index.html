Welcome to STL Manager.


Hello, friend. STLMan is here to help you manage and run your 3D printing stuffs.


Your Navigation:

  • Home (that's here).
  • Longlist will return a page with every entry in your database (use with caution, intended for small databases).
  • Paginate will display a list of entries in smaller lists (more suitable for larger databases).
  • Search will return a long list of every entry that matches your query. Can be long.
  • Update Database will run the database update script

Note that this application uses CURA slicer and will require you have it installed.



Developed by Ian Hill, 2024-2025 with contributions from Connie Hill and ChatGPT.
